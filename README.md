# **Google Cloud 生成式 AI 认证备考全攻略 (中文精编版)**

欢迎来到 **Google Cloud GenAI 认证备考仓库**！本仓库致力于为正在准备 Google Cloud Generative AI 相关认证（如 Cloud Digital Leader, ML Engineer 附加模块或专门的 GenAI 认证）的开发者、架构师和业务决策者提供**高质量、结构化、中英对照**的复习资料。

所有资料均经过深度提炼与排版优化，去除了冗余信息，直击核心考点。

## 仓库内容 (Repository Contents)

本仓库目前包含两个核心复习文件：

### 1. [知识点速记指南 (Cheat Sheet)](./genai_cheat_sheet.md)

这是您的**考前冲刺核心讲义**。笔者将庞杂的官方文档提炼成了极其精简的考点清单，按主题分类，重点加粗。

* **涵盖内容**：深度学习与基础模型概念、Vertex AI 等 Google Cloud 核心工具链、RAG 与提示工程技巧、SAIF 安全框架、Responsible AI 原则及企业落地战略（如创意矩阵法）。

* **使用建议**：建议在刷题前通读此文档，构建完整的 GenAI 知识体系。

## 核心考点大纲 (Syllabus Covered)

本仓库的资料严格按照以下四个核心主题进行组织：

1. **主题 1：生成式 AI 基础 (GenAI Fundamentals)**

   * AI、ML、DL 与基础模型的层级关系。

   * 强化学习、无监督学习等机器学习核心。

   * 大语言模型 (LLM) 与扩散模型 (Diffusion Models) 的原理。

   * 模型的局限性（幻觉、数据依赖、知识截点等）。
   
   * **安全 AI (Secure AI)**：SAIF 框架、三大安全威胁（数据中毒、模型盗窃、对抗性攻击）及防御。
   
   * **Responsible AI 核心原则**：安全为本、透明度、隐私、公平性、可解释性 (Explainable AI) 及法律合规。

2. **主题 2：Google Cloud 解决方案 (Google Cloud Solutions)**

   * 核心平台：Vertex AI, Model Garden, Google AI Studio。

   * 特定场景应用：Agent Assist, Vertex AI Search, Document AI, NotebookLM。

   * ML 运维生命周期与 Google Cloud 数据套件（BigQuery, Pub/Sub 等）的映射。

3. **主题 3：AI 开发与提示工程 (AI Development & Prompt Engineering)**

   * 检索增强生成 (RAG) 与数据接地 (Grounding)。

   * 提示词技巧：角色提示 (Role Prompting)、思维链 (Chain-of-thought)、小样本提示 (Few-shot)。

   * 模型参数控制（如输出长度限制）与人机协同 (HITL)。

4. **主题 4：生成式 AI 战略**

   * 企业 AI 战略规划（多向协同策略、创意矩阵法 Idea Matrix）。

   * 各层级管理者的角色与自动化场景界定。

# Study Guide for Google Cloud Generative AI Leader Certification

本项目聚焦于 Google Cloud Generative AI Leader 认证考试，旨在为考生提供全面的学习资料、知识速查表及官方参考文档，助力高效备考与能力提升。

## 目录结构

- `Cheat Sheet/` — 领袖认证核心知识速查表，覆盖考试重点、流程要点、典型场景和案例分析，方便考前查漏补缺。
- `Official Document/` — 官方速记文档
- `LICENSE` — 开源协议说明。

## 谁适用这份指南？

- 正在准备 Google Cloud Generative AI Leader 认证考试的学员
- 计划系统梳理和掌握 Google Cloud 生成式 AI 产品、解决方案及管理理念的 AI Leader、技术主管与项目决策者
- 企业 AI 战略制定/落地、团队管理相关岗位的同仁

## 使用方法

1. 备考时间充足的小伙伴，建议先通览 `Cheat Sheet/`，把握 GenAI Leader 认证考试的整体框架；
2. 备考时间匆忙或者只想快速复习一遍知识点的小伙伴，建议直接阅读 `Official Document/` 中的官方速记文档；
3. 可灵活结合个人学习规划，查漏补缺、融会贯通。
